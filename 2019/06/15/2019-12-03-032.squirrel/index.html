MacOS上的98五笔

  • 百度输入法 MacOS 版

梅花点讯输入法团队被百度收购后，出品了备受好评的「百度手机输入法」，有 iOS , Android , MacOS 三种版本。
在我们98五笔团队的影响下，经由百度法务部查清「98五笔方案」已无 版权 问题后，在 2015 年全平台内置98五笔方案。

百度五笔 windows 版，因在其制作过程中整个团队被强制裁撤掉，无缘内置98五笔方案，并且 windows 版百度五笔已经项目废止，停止了维护。

而继续活着的原梅花点讯团队，在他们维护的 3 种平台上，继续维护着98五笔方案。

  • 清歌输入法 MacOS 版

清歌输入法和98五笔团队一直有良好的沟通，很早就支持了98五笔方案。现在清歌输入法已经非常成熟，内置的98五笔方案为我们98五笔用户广泛使用的「至善98码表」，非常推荐使用。

  • 落格输入法 MacOS 版

落格是一款很不错的输入法工具，作者更新也比较快，与98五笔团队保持着良好沟通，内置「至善98码表」，安装即用。落格输入法是作者的主业，高阶设置版需要付费，如果对落格使用比较满意，建议支持一下。

  • 鼠须管98五笔专版

中州韵在 MacOS 上的前端，叫作「鼠须管」。它已是声名卓著的「神级输入法」。我们当然不会错过它，我们在 Librime 内核所允许的最大范围里，充分挖掘 Librime 的各种可能性，编译制作了「鼠须管98五笔专版」。

方案

我们不仅内置了「单字」和「含词」两种方案。甚至还包括了学术或其他专业用途的「超集」方案。

  • control + ~ or F4 = 方案切换

  • control + shift + h = 拆分显隐开关

  • control + shift + f = 繁简转换开关

  • control + shift + u = 字符集过滤

  • 「z」键引导拼音反查

  • 「`」键引导精准造词

  • 「~」键引导以形查音

重新部署

每次修改参数后，需要「重新部署」以使程序接收你的变更。
98五笔专版的鼠须管，已经由98五笔的高手们优化调教到了最佳状态，如无必要，不建议修改。
