使用 Jekyll 搭建静态博客时，很多操作是摸着石头过河。
备忘如下：

  • 添加了 Mathjax 支持
  • 添加了高亮语法显示支持
  • 自定义了语法高亮细节，包括颜色和字体
  • 在 post 的 css 中添加了code 子项
  • GEM 扩展功能

在 GEM 扩展这里，搞了很多小飞机，没有记录下来，我都弄不清楚了。
留下 GEM 本地目录，以后再搞飞机，可以对照着来。
