柚子·小甜版

发表于2019-12-03|更新于2019-12-03|柚子输入法
|字数总计:679|阅读时长:2分钟|阅读量:

基本情况

  • Win操作系统
  • 98五笔方案
  • 绿色便携软件
  • 支持十万超集
  • 支持划译为98五笔拆解
  • 设置入口齐全

下载地址

98五笔资源库


使用方式

下载后，解压到 U 盘中。进入主程序目录，右键「以管理员权限运行」。

  • 「以管理员权限运行」


  • 「悬浮色块--右键--各种设置项」


特别说明

  • 超集需要到「98五笔资源库」做一下「字体支持」。
  • 如有「拆分显示」需要，设置字体为「98WB-0」或「98WB-P0」。
  • 右键「以管理员权限运行」，可以避免一些「兼容性问题」。
  • ALT + Q = 划译
  • CTRL + Q = 查看字根图
  • 分号、引号 = 2，3 选重

功能引导

〔 / 〕引导〔 标签别名 〕，进入相关〔 功能 〕。比如：

  • 〔 /cj 〕=〔 98超集方案开关 〕
  • 〔 /ci 〕=〔 单字/含词切换 〕

〔 / 〕同时引导〔 符号 〕，与 RIME 保持一致的逻辑：详情

〔 z 〕引导〔 临时拼音兼反查 〕，同时提示〔 历史输入 〕。

〔 ~ 〕引导〔 以形查音 〕。

〔 ` 〕引导〔 快捷符号 〕。

外观典例

设为「98WB-0」字体 && 「更多设置-拆分显示」&&「默认配色」


词库管理

  • 「含词码表」支持「导出」后修改，「导入」后生效。
  • 「单字」&「超集」是固化的，这是「98五笔」的基础表单，不支持更改。
  • 「SQLite」二进制码表，性能无忧。
  • 「快捷造词」已支持。
  • 「在线调频」已支持。
  • 「在线删词」已支持。

好用到哭

  • 开放的码表管理策略。
  • 音形互通，以形查音，以音查形。
  • 拆分提示，五笔划译。
  • 高性能，sqlite 二进制码表，日常占用 10M 以下。
  • 兼容性好，通杀一切 win 发行版。
  • 绿色便携，放到 U 盘，随时随地装逼。
